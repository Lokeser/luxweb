# Testes de Defesa

Testes de Defesa são usados para resistir ou evitar efeitos negativos.

## Exemplos de Defesa por Atributo

- Agarrão → Força  
- Encantamento → Inteligência  
- Efeitos Mentais Estranhos → Sabedoria  
- Explosões em Área → Destreza  
- Doenças → Constituição  
- Efeitos de Mana → Mana  

## Mecânica

O teste é realizado da seguinte forma:

`DR (Dado de Rank) + Atributo + ER (Eficiência de Rank)`

A CD é o valor mínimo necessário para evitar ou resistir ao efeito.
